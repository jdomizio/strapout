Dropdown

A binding to handle the toggling of a bootstrap dropdown menu

Basic Example

This shows the default behavior for the binding and viewModel. The binding is applied to the element that is to trigger the dropdown.

Binding Code

<nav class="navbar navbar-default">
        <div class="container-fluid">
            <div class="navbar-header">
                <div class="navbar-brand">Dropdown Binding</div>
            </div>
            <div class="navbar-collapse collapse">
                <ul class="nav navbar-nav">
                    <li class="dropdown" style="cursor: pointer;">
                        <a class="dropdown-toggle" role="button" data-bind="dropdown: simple">
                            Dropdown Menu
                            <span class="caret"></span>
                        </a>
                        <ul class="dropdown-menu" role="menu">
                            <li role="presentation"><a role="menuitem">Item one</a></li>
                            <li role="presentation"><a role="menuitem">Item two</a></li>
                        </ul>
                    </li>
                </ul>
            </div>
        </div>
    </nav>

ViewModel

this.simple = new Dropdown();

Code-Configuration

There are no options to pass to a bootstrap dropdown that would require configuring in code.

Inline Configuration

If passed an observable instead of an instance of Dropdown, the binding will assume that the observable is meant to control whether the dropdown is open. (Dropdown.isOpen)

Dynamic Behaviors

Dropdown can be opened or closed in code.

NOTICE

The data-api events for a bootstrap dropdown will cause the dropdown to close if you tie the opening of the dropdown to another click event. If this is something you need to do, you may either want to consider a different dropdown menu, or disable the data-api. That's why this demo is using a hover instead of a click.

Notice that if you click the button while the dropdown is showing, it will close. This is expected because we have not disabled the data-api.

Binding Code

<div class="panel panel-default">
    <div class="panel-body">
        <div class="col-xs-6">
            Value of <code>dynamicDropdown.isOpen</code>: <span data-bind="text: dynamicDropdown.isOpen"></span>
        </div>
        <div class="col-xs-6 text-right">
            <button class="btn" data-bind="event: mouseEvents">Hover over me</button>
        </div>
    </div>
</div>
<nav class="navbar navbar-default">
    <div class="container-fluid">
        <div class="navbar-header">
            <div class="navbar-brand">Dropdown Binding</div>
        </div>
        <div class="navbar-collapse collapse">
            <ul class="nav navbar-nav">
                <li class="dropdown" style="cursor: pointer;">
                    <a class="dropdown-toggle" role="button" data-bind="dropdown: dynamicDropdown">
                        Dropdown Menu
                        <span class="caret"></span>
                    </a>
                    <ul class="dropdown-menu" role="menu">
                        <li role="presentation"><a role="menuitem">Item one</a></li>
                        <li role="presentation"><a role="menuitem">Item two</a></li>
                    </ul>
                </li>
            </ul>
        </div>
    </div>
</nav>

ViewModel

this.dynamicDropdown = new Dropdown();
this.mouseEvents = {
    'mouseover': this.dynamicDropdown.open.bind(this.dynamicDropdown),
    'mouseleave': this.dynamicDropdown.close.bind(this.dynamicDropdown)
};
