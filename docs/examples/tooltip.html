Tooltip

A binding to display a bootstrap tooltip

Basic Example

This shows the default behavior for the binding / viewModel.

Demo

Binding Code

<button class="btn" title="tooltip text" data-bind="tooltip: simpleTooltip">Hover Over Me!</button>

ViewModel

this.simpleTooltip = new Tooltip();

Code-Configured Tooltips

Options for displaying the tooltip can be configured within your viewModel

Demo

Binding Code

<button class="btn" data-bind="tooltip: codeConfiguredTooltip">Hover Over Me!</button>

ViewModel

this.codeConfiguredTooltip = new Tooltip({
    title: 'I was configured with code!',
    options: {
        placement: 'right'
    }
});

Dynamic Behaviors

You can change the state of the observable properties of the tooltip to have it reflect in your view

Demo

To show tooltip you can hover over me or press the 'toggle' button.

Binding Code

<div class="panel" data-bind="tooltip: behaviorTooltip">
    <h3>You can hover over me or press the 'toggle' button.</h3>
</div>

ViewModel

this.behaviorTooltip = new Tooltip();
this.behaviorTooltip.options.placement = 'bottom';
this.behaviorTooltip.title('Default messages are awesome!');

NOTICE

The value displayed in the tooltip will not change until the tooltip has been hidden and shown again. This is in line with the behavior that is presented in Bootstrap

Inline Configuration

You can also define options in the tooltip binding

Demo

Binding Code

<button class="btn" data-bind="tooltip: inlineTooltip,
                               tooltipOptions: { title: 'Configured in binding', placement: 'left', trigger: 'click' }">Click Me!</button>

ViewModel

this.inlineTooltip = new Tooltip();
